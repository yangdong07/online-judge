
### 读优化

cin、cout 的效率不如 scanf 和 printf。

但是在大量读的情况下，可以使用读优化，可以大大减少评测时间。

注意读优化有一定的限制，比如只能读取大于0的 `int` 型数字，下面是读优化代码的基本模板。

```cpp
int read_num()
{
    char c;
    int num = 0;
    for (c = getchar(); c < '0' || c > '9'; c = getchar());

    for (; c >= '0' && c <= '9'; c = getchar())
        num = num * 10 + c - '0';
    return num;
}
```
