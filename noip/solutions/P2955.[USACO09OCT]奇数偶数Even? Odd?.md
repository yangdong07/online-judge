### P2955 [USACO09OCT]奇数偶数Even? Odd?

其他标签: **USACO**, **2009**


#### Description

#### Analysis

此题主要考察

1. 整型范围
2. 如果用整型不能表示一个数字，用字符串。

一开始以为是水题。

后来发现，数的范围是 $10^{60}$， `long long` 才 $2 ^ {64}$ 。简直变态。

如果用整型不能表示一个数字，用字符串。

```cpp
int main()
{
    int n, i, len;
    char s[MAX_SIZE];
    scanf("%d", &n);
    for (i = 0; i < n; ++i)
    {
        scanf("%s", s);
        len = strlen(s);
        printf((s[len - 1] - '0') & 1 ? "odd\n" : "even\n");
    }

}
```
