### P1075 质因数分解

算法标签: **数论,数学**, **素数判断,质数,筛法**
其他标签: **NOIp普及组**, **2012**


#### Description

已知正整数 $n$ 是两个不同的质数的乘积，试求出两者中较大的那个质数。

#### Analysis

这题其实很水。 题目指定了 $n$ 只能分解成两个不同的质数。。。除此之外不可能有别的质数分解， 也不可能有别的因数分解。 那就挨个数看看能不能整除就行了： 因为 整除 即 唯一的质因数分解。

```cpp
int main() {
    int n;
    scanf("%d", &n);
    for (int i = 2; i < n; ++i) {
        if (n % i == 0) {
            printf("%d", n / i);
            return 0;
        }
    }
}
```
