#### Analysis

这题主要证明一个式子就可以了。以后也会用到。

如何求  `ceil(1.0 * n / m)` ，而不是通过浮点运算方式。

一种写法是 `(n - 1) / m + 1`

分几种情况：  

- `m = 1` 不用证了。
- `m > 1`，
    - 如果 n 能被 m 整除，那么 `(n - 1) / m == n / m - 1`
    - 如果 n 不能被 m 整除，就是余数不为 0 ，那么 n 减掉 个 1 没什么影响，整数部分都一样。 `(n - 1) / m == n / m`。
    - 所以 m 不能整除 n： `(n - 1) / m + 1 == n / m + 1`
    - 所以 m 能整除 n： `(n - 1) / m + 1 == n / m`
