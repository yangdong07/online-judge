### P1147 连续自然数和

算法标签: **前缀和**, **数论,数学**, **枚举,暴力**


#### Description

#### Analysis

简单的数学题。

设有这么一个区间 `[a, b]`，区间和为 $(b + a)(b - a + 1) / 2 = n$

如果 $2n = pq$，并且 $p < q$，那么有

$a=(q - p + 1) / 2, b =(q +p -1)/2$，并且要求 $p+q$ 为奇数。

如果 $p,q$ 相差越小， 则 $a$ 越小。 所以有以下代码：

```cpp
int main()
{
    int n, i;

    cin >> n;

    n <<= 1;  // n *= 2
    for (i = (int)sqrt(n); i > 1; --i)
    {
        if (n % i == 0 && (n / i + i) & 1)
            cout << (n / i - i + 1) / 2 << " " << (n / i + i - 1) / 2 << endl;
    }
    return 0;
}
```

没什么可以优化的。有也懒得弄。
